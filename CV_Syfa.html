    • Di Departemen Teknik Geodesi sendiri (klik link ini untuk menuju website resmi Teknik Geodesi) terdapat salah satu mata kuliah yaitu SigWeb. Praktikum SigWeb secara keseluruhan membahas mengenai pemrograman menggunakan html, css, dan javascript. Dalam praktikum ini diajarkan cara menggunakan html, css, dan js sehingga menghasilkan suatu output tertentu. Pada minggu-minggu awal masih berupa pengenalan mengenai Sistem Informasi Geografis berbasis web. Sedangkan mulai minggu keempat dikenalkan berupa bootstrap. Di Minggu ke-5 dikenalkan berupa leaflet dan GeoJson. Leaflet merupakan library atau kumpulan fungsi berbasis Javascript yang digunakan untuk menampilkan peta interaktif pada halaman web.GeoJSON dapat digunakan untuk menampilkan data spasial pada media web. LeafletJS dapat memanggil layer GeoJSON sebagai variable dan ditambahkan pada peta. Ouput hingga minggu ke -7 berupa leaflet yang memuat informasi berupa titik atau poligon dalam sebuah peta.
